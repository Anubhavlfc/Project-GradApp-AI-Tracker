           EMAIL INTEGRATION SETUP - QUICK CHECKLIST
================================================================================

TIME REQUIRED: ~20 minutes (one-time setup)

--------------------------------------------------------------------------------
STEP 1: GET GMAIL API CREDENTIALS (15 minutes)
--------------------------------------------------------------------------------

1. Go to: https://console.cloud.google.com/

2. Create project: "GradTrack-AI-Email"

3. Enable Gmail API:
   - Search: "Gmail API"
   - Click "Enable"

4. Configure OAuth Consent Screen:
   - Go to: APIs & Services ‚Üí OAuth consent screen
   - Select "External"
   - App name: GradTrack AI
   - Add your email
   - Scopes: gmail.readonly
   - Test users: Add your Gmail address

5. Create Credentials:
   - APIs & Services ‚Üí Credentials
   - Create Credentials ‚Üí OAuth client ID
   - Type: Desktop app
   - Name: GradTrack AI Desktop

6. Download credentials:
   - Click download icon (‚Üì)
   - Save as: backend/credentials.json

--------------------------------------------------------------------------------
STEP 2: GET ANTHROPIC API KEY (2 minutes)
--------------------------------------------------------------------------------

1. Sign up: https://console.anthropic.com/

2. Create API key:
   - Go to: API Keys
   - Click "Create Key"
   - Copy the key (starts with sk-ant-...)

3. Create .env file:
   cd backend
   echo "ANTHROPIC_API_KEY=sk-ant-your-key-here" > .env
   
   (Replace sk-ant-your-key-here with your actual key!)

--------------------------------------------------------------------------------
STEP 3: VERIFY SETUP
--------------------------------------------------------------------------------

Run this to check everything:

cd /home/user/Project-GradApp-AI-Tracker
./setup_email.sh

You should see:
‚úÖ Gmail credentials: Ready
‚úÖ Anthropic API key: Ready
üéâ You're all set!

--------------------------------------------------------------------------------

--------------------------------------------------------------------------------

Terminal 1 - Backend:
cd backend
python main.py

Terminal 2 - Frontend:
cd frontend
npm run dev

Browser:
http://localhost:3000

Click "Email Sync" button ‚Üí Connect Gmail ‚Üí Scan ‚Üí Import!
